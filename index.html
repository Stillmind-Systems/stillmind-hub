🚀 Ce que je construis

Je développe des outils avancés dans l’écosystème open source, principalement en Go, avec un focus sur :

  • 🧠 Systèmes cognitifs pour développeurs — Extensions VS Code (StillMind Vault, DevContext Pro, Logic Time Machine) qui introduisent mémoire contextuelle et reconstruction logique du code
  • ⚙️ IA locale & autonome — Déploiement de modèles via Ollama combinés à ChromaDB pour créer des systèmes capables de mémoriser et raisonner localement
  • 🧩 Architectures intelligentes — Conception de systèmes où l’IA agit comme une extension cognitive du développeur

🔬 Recherche & Exploration

  • 🤖 Physics-Informed AI — Intégration de contraintes physiques dans les modèles d’apprentissage
